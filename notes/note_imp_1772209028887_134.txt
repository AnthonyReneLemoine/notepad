__**Configurer correctement son jeu / Pourquoi configurer [full-low] c’est le mal sur Shadow?!**__

Afin de configurer correctement son jeu sur Shadow, il faut exploiter à 100% le GPU Nvidia et réduire autant que possible ce qui consomme du CPU (un GPU qui s’ennuie ou calcule des opérations trop simples va saturer le CPU en demandant plus de travail et créer ce qu’on nomme un « CPU Bottleneck »). Le but étant d’avoir le CPU le moins chargé possible afin de nourrir le plus rapidement possible le GPU en nouvelles informations. Il faudra aussi fermer navigateurs / applications inutiles pendant une session de jeu et garder ses pilotes graphiques le plus à jour possible.

**Les options à privilégier consommant du GPU:**

- Résolution (Higher is better ?!)
- Résolution d’affichage -> 100%
- Multithreaded Rendering -> On
- DirectX 12/Vulkan 
- Textures/Terrain/Material/Effect Quality : le plus haut possible (Medium/High/Ultra)
- Anisotropic Filtering (AF) : le plus haut possible (Medium/High)

**Les options à réduire consommant du CPU:**

- Ombres (Shadow) -> Off/Low/Medium
- Eclairage dynamique (Dynamic Lighting) -> Off
- Particules -> Off/Low/Medium
- Draw Distance -> Plus bas possible
- (Anticrénelage (Anti-Aliasing / AA) voir "cas par cas")
- Motion Blur -> Off/Low
- Water Reflection -> Off
- Physics (Hormis Jeux Nvidia PhysX)

**Les options à tester au cas par cas:**

- DLSS / Lens Quality -> On/Off
- Shader Quality -> Low/High
- Plein écran sans bordures (Borderless Fullscreen) / Plein Ecran (Fullscreen)
- Anticrénelage (Anti-Aliasing / AA) dans l’ordre TXAA -> MSAA -> FXAA -> SSAA / Off
- Adaptive V-Sync dans le Nvidia Panel

**Les options à désactiver:**

- Planification GPU (GPU Scheduling) - (Win 10/11) / HDR -> Incompatible Shadow
- Mode Jeu Windows -> Baisse de performance
- Son AC3 / DTS / Atmos / Spatialisation -> Incompatible Shadow
- V-Sync, GPU Scaling <100% OU >100% -> Baisse de performance / Input lag
